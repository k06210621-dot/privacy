PredictX Sports 隱私權政策

最後更新：2026-06-29

感謝您使用 PredictX Sports（以下簡稱「本 App」）。本隱私權政策說明本 App 如何蒐集、使用、儲存及保護您的資料。請您在使用前詳閱以下內容。

1. 我們蒐集的資料

本 App 為純資訊提供 App，不接受投注、不銷售預測商品。App 本身不追蹤、不記錄使用者的點擊行為或瀏覽歷史。我們僅蒐集以下技術性識別資訊：

  • APNs 推播識別碼：您開啟推播通知時，系統產生的 device token（用於發送高信心度賽事推播；您可隨時在 App 內關閉推播）
  • 訂閱資訊：透過 Apple StoreKit 處理的訂閱等級（Free / Basic / Standard / Premium），交易紀錄由 Apple 處理，本 App 不直接儲存信用卡資訊
  • 廣告識別碼 (IDFA)：僅在您同意時用於 Google AdMob 個人化廣告（您可隨時在 iOS 設定 → 隱私權 → 追蹤中關閉）

本 App 不會蒐集以下資料：

  • email、姓名、Apple ID 等個人識別資訊（無註冊或登入機制）
  • 地理位置資訊
  • 聯絡人、照片、裝置儲存內容
  • iOS 版本、機型等裝置資訊
  • App 內點擊行為、瀏覽歷史、解鎖紀錄等使用模式（僅儲存於裝置本地，不上傳至伺服器）

2. 資料用途

  • 提供每日球類賽事（MLB / NPB / CPBL / NBA）AI 推論內容
  • 管理會員訂閱（Free / Basic / Standard / Premium）與恢復購買功能
  • 發送高信心度賽事推播通知（Premium 用戶限定，可隨時關閉）
  • 提供 Google AdMob 廣告（您可透過系統設定關閉）

3. 第三方服務

本 App 使用以下第三方服務：

  • Apple StoreKit：處理訂閱付款
  • Google AdMob：顯示廣告（包含 Google User Messaging Platform 處理同意聲明）
  • 雲端後端 API：提供賽事資料與 AI 推論（位於 predictx-sports-production.up.railway.app）

這些服務有其各自的隱私權政策，本 App 不對其內容負責。

4. 資料儲存與安全

所有資料透過 HTTPS 加密傳輸。後端僅儲存 APNs device token 與訂閱等級（用於推播分眾與管理訂閱），不使用任何使用者行為追蹤技術。裝置端偏好設定（收藏的隊伍、深色模式偏好、解鎖的賽事）僅儲存於您的 iOS UserDefaults，不會上傳至伺服器。App 透過 Google AdMob 顯示的廣告可能涉及跨裝置追蹤，您可隨時在 iOS 設定中關閉。

5. 資料保留期限

  • APNs 推播 token：保留至您解除安裝 App 或手動關閉推播
  • 訂閱等級：保留至您解除安裝 App 或取消訂閱
  • 裝置端偏好設定：隨時可透過解除安裝 App 清除
  • 本 App 不儲存任何使用行為歷史

6. 您的權利

您有權：

  • 查詢、修改、刪除您的資料（透過 iOS 設定或聯絡我們）
  • 取消訂閱（透過 iOS 設定 → Apple ID → 訂閱）
  • 關閉個人化廣告（透過 iOS 設定 → 隱私權 → 追蹤）
  • 隨時關閉推播通知（透過 App 內設定或 iOS 設定）

7. 兒童隱私

本 App 不針對 13 歲以下兒童。我們不會刻意蒐集 13 歲以下兒童的個人資料。如發現有此情況，請聯絡我們刪除。

8. 政策變更

本政策可能不定期更新。重大變更時會於 App 內公告。繼續使用本 App 即視為接受新政策。
